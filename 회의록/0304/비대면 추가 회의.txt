메타버스-햅틱 디바이스를 활용한 여러가지 프로젝트를 생각해봄.

1. 온라인 쇼핑 플랫폼 ->  옷 정보를 확인하고, 햅틱 디바이스 적용해서 촉감으로 옷 질감을 확인 할 수 있음
=> 촉감인식 햅틱 디바이스에 대한 구현 가능성 의문 

2. 시각장애인을 위한 스마트옷장 -> 기온 확인, 옷의 색깔, 질감 확인 가능
=> 햅틱 디바이스와 상관없이 캠으로 옷 인식하여 옷 고르기에 도움을 줌

3. 수화를 음성언어로 자동전환하는 프로그램( 청각장애인과 비청각장애인의 소통) 
=> 수화 인식 장갑 구현이 어려울 것 같음

4. 장애인 학습 프로그램 (ex 수학 그래프 만지기…) 

6.운동(탁구)/ 게임(벽돌깨기/캐치마인드…)
=> 햅틱 디바이스 진동을 통해 벽돌 깰 때 마다 진동

+)추가로 햅틱 디바이스 구현 가능성 정도에 대하여 연구원님께 자문을 구함.
